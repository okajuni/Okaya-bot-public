# Okaya-bot-public
Code du bot discord De la team Bros Corps / OKΛYA 

Le code du bot discord officiel de La team OKΛYA [BC] est enfin disponible ! 

pour pouvoir bien installer tout les prérequis voici une sorte de mini tutoriel :

Premièrement si vous le faites tourner sur votre pc pour des test vous allez avoir besoin de node.js voici le lien :  https://nodejs.org/fr/ 

ensuite vous allez avoir besoin de différent packages pour pouvoir lancer le bot 
une fois dans le dossier (et non dans l'archive .zip) vous allez devoir lancer votre invite de commande puis taper : 

## node install discord.js
## node install fs 

______________________________________________________!

SI VOUS AVEZ LE MESSAGE : 
'node' n’est pas reconnu en tant que commande interne
ou externe, un programme exécutable ou un fichier de commandes.

VOUS DEVEZ MODIFIER VOTRE PATH : https://www.youtube.com/watch?v=M2BWTJXDJXY 
_______________________________________________________!

une fois les installation finis vous pouvez lancer le bot avec :  

## node index.js 

Le bot donne dans ses logs beaucoup d'id (utilisateurs, channels, serveurs mais plus souvent d'utilisateurs) 
pour les utiliser je vous recommande (et pas que pour ce bot) d'utiliser ce site : https://discord.id 
qui peut vous permettre par exemple si vous mettez l'id : 478881945020006402 
vous trouverez le nom, photo de profile,les badges et la date de création du compte (et si c'est un bot ou non) 

Pour modifier ou créer du contenue pour le bot premièrement apprenez à utiliser la librérie discord.js : https://discord.js.org/#/docs/main/stable/general/welcome
(si vous avez une version trop récente et que le bot ne marche plus à cause de cela envoyez moi le problème je vous aiderai) 
https://www.youtube.com/watch?v=4Np2fJOdXjE&list=PLdnyVeMcpY7_IiC977keSLBXXJG_I3vUo 

cette playlist est pour moi un bon moyen de comprendre et d'apprendre pour modifier le bot (la version Française existe aussi vous pouvez la chercher)
